<h1>Microsoft Azure - How To Create And Configure A New User Account</h1>


<h2 style="font-family: Arial, sans-serif; font-size: 20px; font-weight: bold; margin-top: 24px; margin-bottom: 12px;">
⏹️ Description</h2>

<p style="font-family: Georgia, serif; font-size: 16px; margin-top: 12px; margin-bottom: 12px;">
This project required a structured and secure way to manage internal user identities and collaborate with external participants using a centralized cloud-based identity platform. The responsibility was to create, configure, and validate internal user accounts and invite an external user within Microsoft Entra ID to ensure proper identity management and access readiness. Microsoft Entra ID was accessed through the Azure portal to create a new internal user with defined identity attributes such as role, department, and usage location, followed by generating secure credentials and validating account creation. An external user was then invited using the built-in guest invitation workflow, including profile configuration and automated email notification to support external collaboration. The environment was successfully prepared with both internal and external identities properly configured, enabling secure access management and demonstrating effective use of Microsoft Entra ID for user lifecycle administration.
