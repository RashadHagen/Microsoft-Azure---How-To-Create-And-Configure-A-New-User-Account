<h1>Microsoft Entra ID - How To Create And Configure A New User Account</h1>


<h2 style="font-family: Arial, sans-serif; font-size: 20px; font-weight: bold; margin-top: 24px; margin-bottom: 12px;">
⏹️ Description</h2>

<p style="font-family: Georgia, serif; font-size: 16px; margin-top: 12px; margin-bottom: 12px;">
In this project, a standardized identity is created in Microsoft Entra ID to support controlled access, role assignment, and user management within an Azure-based environment. You will create and validate a new user account in Microsoft Entra ID with appropriate properties to ensure the account is enabled, identifiable, and ready for administrative or lab use. You will use the Azure portal to navigate to the Users management blade, initiated the Create new user workflow, and configured required identity and profile properties such as user principal name, display name, job title, department, usage location, and account status while leveraging auto-generated credentials. You will complete the review process and finalized creation, then refreshed the directory to confirm successful provisioning. The user account will be successfully created, enabled, and visible in Microsoft Entra ID, providing a ready-to-use identity.
